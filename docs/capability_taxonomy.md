# Capability Taxonomy

## Core LLM Capabilities

- Answer accuracy: factual QA, reference matching, category/domain profiles.
- Truthfulness: correction of misconceptions and avoidance of false references.
- Instruction following: output format, constraints, length, JSON validity, style.
- Reasoning and logic: arithmetic, symbolic reasoning, multi-hop reasoning,
  contradiction detection, and final-answer correctness.

## System Capabilities

- RAG capability: context use, groundedness, completeness, retrieval hit rate,
  faithfulness, citation support.
- Tool use: tool choice, argument validity, unnecessary calls, sequence quality,
  recovery from tool errors.
- Agentic task completion: planning, execution, validation, task success, trace
  completeness, cost and latency tradeoffs.
